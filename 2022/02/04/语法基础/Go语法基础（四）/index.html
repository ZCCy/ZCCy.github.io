Go语法基础（四）：更多特性

1.strings

  • strings.Fields()

    返回值：str的子字符串切片或如果str仅包含空格的空白切片。也就是返回值是切片

    例：

    v:= strings.Fields(“go is good”)

    输出为[go is good]

    strings.Fields("go\nis\tvery \n fun")

    输出为 [go is very fun]

2.函数值

  • 函数也是值。它们可以像其它值一样传递。函数值可以用作函数的参数或返回值。

  • 函数内不能声明函数,只能创建一个匿名函数

  • 如果返回值只有一个参数，并且没有名字，那么不需要加圆括号

3.函数的闭包

    func add(base int) func(int) int {

    base += i
    return base

4.Printf

  1. %v 只输出所有的值
  2. %+v 先输出字段类型，再输出该字段的值
  3. %#v 先输出结构体名字值，再输出结构体（字段类型+字段的值）

5.目录与包

  • 大写字母开头的函数相当于 public,小写字母开头的函数相当于 private, 只有相同包名的文件才能使用
  • 在go语言中，同一层级目录，不允许出现多个包名
